Comandos CLI Moovsec

  • ls

    List directory contents.

    Listar o conteúdo do diretório.

  • cd

    Change the current directory.

    Altere o diretório atual.

  • mkdir

    Create a new directory.

    Crie um novo diretório.

  • rm

    Remove files or directories.

    Remova arquivos ou diretórios.

  • top

    Show real-time system processes.

    Mostre processos do sistema em tempo real.

  • grep

    Search for patterns in files.

    Procure padrões em arquivos.

  • moovsec help [comand]

    View on-screen help for the command.

    Vizualiza a ajuda do comando na tela.

  • moovsec auth reset moovsec

    Recovers Moovsec administrator password.

    Recupera a senha de administrador do Moovsec.

  • moovsec auth reset server-manager admin

    Recovers Server Manager administrator password.

    Recupera a senha de administrador do Server Manager.

  • moovsec config list [nome]

    List name configuration [name].

    Lista a configuração de nome [nome].

  • moovsec config update [nome] [valor]

    Updates a configuration named [name] to have the value [value].

    Atualiza uma configuração de nome [nome] para possuir o valor [valor].

  • moovsec status

    Lists server and Moovsec status and monitoring data.

    Lista dados de status e monitoramento do servidor e do Moovsec .

  • moovsec status moovsec

    Lists Moovsec status and monitoring data.

    Lista dados de status e monitoramento do Moovsec.

  • moovsec status system

    Lists system status and monitoring data.

    Lista dados de status e monitoramento do sistema.

  • moovsec version

    Allows you to view the CLI version.

    Permite visualizar a versão da CLI.

  • wget [URL] (pode ser necessário ter que configurar o DNS da máquina antes)

    .

    Download do arquivo do Cronos direto no servidor.

  • gzip -d -c [nome do arquivo] | tar -xv ou tar -xf [nome do arquivo] ou tar -xzvf [nome do arquivo]

    .

    Extração dos arquivos de instalação/atualização (1.4.1 ou superior).

  • find / -name listCurrentConfigs.sh

    .

    Verificar diretório de instalação do MoovSec.

  • ufw status (página com mais comandos do UFW)

    .

    Verificar status do firewall UFW do Linux.

  • Verificar status do firewall Firewalld do Linux

    .

    systemctl status firewalld.service (página com mais comandos do Firewalld).

  • grep avx /proc/cpuinfo ou lscpu | grep avx data-category="linux"

    .

    Verificar se há suporte AVX na CPU.

  • htop

    .

    Verificar consumo dos processos em tempo real.

  • cat /etc/os-release

    .

    Verificar SO.

  • lscpu

    .

    Verificar CPU.

  • cat /proc/meminfo

    .

    Verificar RAM.

  • df -h .

    .

    Verificar infos do disco da pasta atual.

  • docker logs [nome do serviço]

    .

    Verificar logs de algum serviço do MoovSec.

  • docker logs moovsec_iot_handler -f -n 300 | grep [ID do dispositivo]

    .

    Verificar logs do iot_handler para verificar comunicação com o ID do MDVR.

  • docker ps -a

    .

    Listar os containers.

  • docker remove [ID do serviço]

    .

    Apagar o container.

  • docker restart [nome do serviço]

    .

    Reiniciar apenas um serviço específico.

  • netstat -tuln | grep : [porta]

    .

    Verificar se certa porta está em uso na rede.

  • docker compose --env-file .env -f ./moovsec-docker-compose.yml down

    docker compose --env-file .env -f ./moovsec-docker-compose.yml up -d

    .

    Recriar todos os containers do MoovSec (dentro do diretório de instalação).

  • docker exec -it moovsec_mongodb mongosh moovsec

    .

    Acessar o banco de dados.
